# LMS Bridge — Cost-to-Run & Funding (Free / Community Model)

**Goal:** run LMS Bridge as a free service for the community. This estimates *what each user/visit actually costs you to serve*, and lists the grants, prizes, and credit programs it's realistically eligible for.

**Headline:** Serving this is **cheap** — fractions of a cent for a casual visitor, on the order of **$0.05–0.50 per active student per semester** for AI, on top of a **~$0–15/month** infrastructure floor. And because the platform already supports **bring-your-own AI key**, you can push the entire AI cost onto each institution and run the public service for **essentially the price of hosting alone**. Free researcher/cloud credits can zero out even that.

> Prices below are current as of June 2026. LLM token prices for Anthropic and OpenAI are from their live pricing pages; others (Gemini, DeepSeek, Render/DO) are noted. Token-per-session counts are engineering estimates from how LMS Bridge actually prompts the model — treat them as a model, not a measurement.

---

## 1. What drives cost

Three things, in order of how much they matter:

1. **LLM tokens** — the only real *per-user* variable cost. Generated by (a) interactive tutor-session turns and (b) remediation-module generation. The mock provider is free, so a demo/pilot with `LLM_PROVIDER=mock` costs **$0** in AI.
2. **Hosting** — a fixed monthly floor (backend + database; the React frontend can be free static hosting).
3. **Everything else** (bandwidth, storage) — negligible at community scale.

A "visit" to the marketing site or a demo browse touches static files and a couple of cheap API reads → **well under $0.01**, effectively free. The cost that matters is an **active student doing remediation**.

## 2. Token cost per tutoring session

A full interactive tutor session in LMS Bridge is an opening turn plus ~6–10 back-and-forth turns. Each turn resends the system prompt (concept + objectives + the student's evidence + grounded course-material excerpts) plus the running history. Modeled total per session ≈ **~20,000 input tokens + ~2,000 output tokens** (conservative; prompt-caching cuts the repeated system prompt by ~90% and roughly halves input).

| Model (current cheap options) | Price in / out per 1M | **Cost / session** | With prompt caching |
|---|---|---|---|
| **Claude Haiku 4.5** | $1.00 / $5.00 | **~$0.030** | ~$0.020 |
| **Gemini 2.5 Flash** | $0.30 / $2.50 (est.) | **~$0.011** | ~$0.008 |
| **GPT-5.4-nano** (OpenAI's cheapest) | $0.20 / $1.25 | **~$0.0065** | ~$0.005 |
| **DeepSeek V4 Flash** | $0.14 / $0.28 (est.) | **~$0.0034** | ~$0.003 |
| Claude Sonnet 4.6 (premium quality) | $3.00 / $15.00 | ~$0.090 | ~$0.050 |
| Local/self-hosted open model (Llama/Mistral) | hardware only | ~$0 marginal | — |

Remediation-module *generation* is a single call (~1–2k in, ~0.5k out) → roughly **$0.001–0.005** each. Minor.

**So one session costs about half a cent to three cents** on a sensible cheap model.

## 3. Cost per active student, and at scale

Assume an active student does ~10 tutor sessions a semester plus a handful of generated modules.

| Model | Per active student / semester | 500 students | 5,000 students |
|---|---|---|---|
| Claude Haiku 4.5 | ~$0.30 | ~$150 | ~$1,500 |
| Gemini 2.5 Flash | ~$0.12 | ~$60 | ~$600 |
| GPT-5.4-nano | ~$0.07 | ~$35 | ~$350 |
| DeepSeek V4 Flash | ~$0.04 | ~$20 | ~$200 |

Add the **hosting floor** (same regardless of model, until traffic is high):

| Component | Cheapest viable | Notes |
|---|---|---|
| Frontend (React static) | **$0** | Cloudflare Pages — free, unlimited bandwidth |
| Backend (FastAPI) | **$0–7/mo** | Render free (sleeps when idle) or Render Starter $7, or a $4–6/mo DigitalOcean droplet |
| Postgres | **$0–6/mo** | Neon free (0.5 GB, auto-suspend) or Render Basic-256mb $6 |
| **Infra total** | **~$0–13/mo** | i.e. **~$0–80 per semester** |

**Bottom line:** a free pilot for **a few hundred students** runs in the ballpark of **$50–230 per semester all-in**, and **thousands of students** is still only a few hundred to ~$1,500/semester on the priciest cheap model — comfortably inside the free-credit programs below.

## 4. The cost-killer you already built: bring-your-own-key

LMS Bridge already supports a **per-institution AI provider + key** (encrypted, with PII-minimization). For a community/freeware model this is the decisive lever:

- **Each institution plugs in its own AI key** → *they* pay for their students' tokens, not you. Your marginal AI cost becomes **$0**, and you only carry hosting (~$0–13/mo). This is also the FERPA-friendly path (their data, their model contract).
- Offer a **shared free tier** (you cover tokens via credits) for individual instructors / small pilots with no institutional key, capped with the **seat limit** you already implemented so a runaway class can't blow your budget.
- Keep **`mock` mode** as the zero-cost demo so anyone can try the full flow for free.

This means "free for the community" is genuinely sustainable: **mock = free, BYO-key = free to you, shared tier = covered by credits/grants.**

## 5. Funding: credits, grants, prizes

Your profile — **free, open-source, NYU-affiliated, higher-ed STEM** — is a strong fit for *researcher credits, open-source grants, and NYU competitions*. It's a **weak** fit for startup-VC credit tiers and SBIR grants, which require a for-profit company. Best bets:

### A. AI / cloud credits (directly cancel your token + hosting cost)
| Program | Amount | Fit |
|---|---|---|
| **OpenAI Researcher Access** | up to **$1,000** API credits / yr | Strong — NYU research affiliation qualifies; reviewed quarterly |
| **Google Cloud / Gemini academic research credits** | **$1,000/yr** (PhD students), up to $5,000 (faculty/postdoc) | Strong if you're a PhD student; covers Gemini |
| **Anthropic External Researcher Access** | free Claude credits | Medium — frame the work as an AI-evaluation research question |
| **Azure for Students** | **$100** + free services | Easy; covers Azure OpenAI |
| **GitHub Student Developer Pack** | dev tools + $100 Azure | Easy with .edu email |
| Gemini API free tier | $0, permanent (rate-limited) | Great for dev/prototyping |

Stacking the OpenAI + Google research credits alone (~$2,000/yr) covers **thousands of students** at the cheap-model rates above — i.e., the public free tier can run on credits indefinitely if you renew them.

### B. Grants (best open-source / academic fits)
| Program | Amount | Fit / caveat |
|---|---|---|
| **NLnet — NGI Zero Commons Fund** | **€5,000–€50,000** | **Best grant fit** — funds individuals/OSS directly, light ~1–2 page application, must be FOSS-licensed, recurring calls |
| **NSF IUSE: EDU** | **$400K**+ / 3 yrs | Strong theme (undergrad STEM), but **NYU is the awardee** — needs a faculty PI (your advisor), not you solo |
| **NSF open-source ecosystems (PESOSE, ex-POSE)** | up to **$1.5M** | For maturing a research tool into a sustainable OSS ecosystem; institution-led |
| **Sloan — Open Source in Science** | up to **$750K** | Routed through an NYU OSPO; funds OSS sustainability more than app-building |
| ED/IES SBIR | $250K → $1M | Funds "AI adaptive tutors" but **requires a for-profit company** (deadline cycle ~June) |

*Out of scope (flagged so you don't waste time): CZI EOSS (biomedical only), Schmidt HAVI (humanities, excludes pedagogy), Mozilla MOSS (on hiatus), AWS Imagine (excludes educational institutions), Overdeck/CSforAll (K-12).*

### C. Prizes / competitions
| Program | Amount | Fit |
|---|---|---|
| **Learning Engineering Tools Competition** (Schmidt Futures / Learning Agency) | up to **$500K** (tiers $50K/$150K/$300K) | **Best thematic prize** — open to students/researchers/educators, non-dilutive. 2026 abstracts closed Oct 2025; target the next cycle |

## 6. Recommendation

1. **Launch free on a near-zero-cost stack now:** Cloudflare Pages (frontend, free) + Render free/Starter or a $4 droplet + Neon free Postgres. Keep **`mock` mode** for demos.
2. **Default institutions to bring-your-own-key** so AI cost sits with them; offer a **credit-funded shared free tier** (seat-capped) for instructors without a key.
3. **Apply now for OpenAI Researcher Access ($1K) + Google research credits ($1K)** to fund that shared tier; add Azure for Students ($100) and the GitHub pack.
4. **Apply to NLnet NGI Zero Commons** (best OSS grant) and the **NYU Prototyping Fund** (immediate), and put the **Learning Engineering Tools Competition** and **NYU Entrepreneurs Challenge** on the calendar.
5. If you want a larger, durable grant, work with an **NYU faculty PI on NSF IUSE: EDU** — the natural home for "AI-guided adaptive remediation in undergraduate STEM."

**Net:** running this free for a real community is affordable (low hundreds of dollars per semester at most, often ~$0 with BYO-key + credits), and the open-source + NYU + STEM framing opens several non-dilutive funding doors that a for-profit version would not.
